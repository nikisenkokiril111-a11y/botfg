# 🎰 CaseMaster — Telegram Bot

Симулятор открытия кейсов с Telegram Stars, мини-играми и полной админ-панелью.

## 🚀 Быстрый старт

### 1. Создайте бота в BotFather

1. Найдите @BotFather в Telegram
2. Отправьте `/newbot` и следуйте инструкциям
3. Скопируйте токен бота
4. Включите оплату Stars: `/mybots` → ваш бот → **Payments** → **Telegram Stars**

### 2. Деплой на Railway.app

1. Зайдите на [railway.app](https://railway.app)
2. Нажмите **New Project** → **Deploy from GitHub repo**
3. Выберите репозиторий с ботом (или загрузите файлы)
4. Перейдите в **Variables** и добавьте переменные:

| Переменная | Значение | Описание |
|-----------|---------|---------|
| `BOT_TOKEN` | `1234567890:ABC...` | Токен от BotFather |
| `ADMIN_IDS` | `123456789,987654321` | Telegram ID администраторов |

5. Нажмите **Deploy** — бот запустится автоматически!

### 3. Локальный запуск

```bash
pip install -r requirements.txt
BOT_TOKEN=ваш_токен ADMIN_IDS=ваш_id python bot.py
```

## 📋 Функционал

### Для пользователей:
- 🎁 **15 кейсов** от 15 до 5000 Stars с системой весов
- 🎀 **Ежедневный кейс** — бесплатно каждый день
- 📦 **Инвентарь** — просмотр и продажа предметов боту (50% стоимости)
- 🎰 **Казино**:
  - 🚀 **Ракета** — коэффициент растёт, выбирай когда забрать
  - 🪙 **Орёл/Решка** — классическая игра x2
- 🎫 **Промокоды** — скидки, бонус звёзд, бонус шанса
- 📤 **Вывод предметов** — заявки администратору

### Для администраторов:
- 👥 **Управление юзерами** — поиск, выдача звёзд, бан/разбан
- 🎁 **Управление кейсами** — изменение цен прямо в боте
- 📤 **Заявки на вывод** — просмотр, одобрение или отклонение
- 🎫 **Промокоды** — создание кодов 3 типов:
  - ⭐ `stars` — выдаёт звёзды на баланс
  - 🎯 `chance` — бонус шанса на 24 часа
  - 💰 `discount` — скидка на покупку
- 📊 **Статистика** — общая статистика бота

## 💳 Система оплаты

Бот использует **Telegram Stars** (XTR) — внутреннюю валюту Telegram.
- Пользователи покупают Stars в Telegram
- Бот принимает Stars через встроенную платёжную систему
- Не требует сторонних платёжных шлюзов

## 🗃️ База данных

Бот использует **SQLite** (файл `cases_bot.db` в папке бота).

На Railway.app база сбрасывается при каждом деплое — для постоянного хранения подключите Railway PostgreSQL или используйте volume mount.

- Перед деплоем обязательно включите **Telegram Stars** в BotFather
